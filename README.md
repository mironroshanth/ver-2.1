# ver-2.1
Temperature Chart
The Temperature Chart website is an open-source digital platform designed to make medical record-keeping simple, accessible, and reliable. Built with healthcare needs in mind, it allows users to easily log, track, and visualize body temperature readings over time. 
Whether used by individuals monitoring their own health, caregivers supporting patients, or medical professionals seeking accurate records, the site provides a streamlined interface that prioritizes clarity and usability. 
The website features a Getting Started guide to help new users quickly learn how to record and manage data, while the About page offers deeper insight into its purpose, vision, and community-driven development.
